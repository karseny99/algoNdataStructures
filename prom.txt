   1) Команды ОС UNIX. Формат команд. Интерпретатор команд.

	 Формат:  команда -ключи -ключи ...  прочие разные аргументы ...
	 Перенаправление вывода:  команда 2> имя_файла
	 Перенаправление ввода:   команда < имя_файла
	 Пере(До-)запись >, >>
	 Примеры команды: pwd, ls, cd, cat, cp, mv, rm, touch, man
	 

	 Интерпретатор командной строки(shell==оболочка) - программа, которая принимает команды от пользователя и исполняет их.
	 Ныне bash.
	 К её функциям относится: Обработка метасимволов, паренаправление ввода/вывода

   2) Файловая система ОС UNIX (структура, именование, навигация).

	 Существует четыре основных типа файлов: 
		обычный файл (regular)  (хранение текста, данные, программы), 
		каталог (directory)    (хранение имен других файлов), 
		специальный (special)   (используются для других устройств - принтеры, терминалы), 
		символическая связь (symbolic link)  (содержат инструкции, которые указывают на другой файл).
	

	  Именование:
		UNIX учитывает к регистрам букв: mydoc != MYDOC
		До 256 бит в UFS файловой системе. 
		Не использовать пробелы и спец. символы в названии.


	 Все файлы являются частью файловой системы UNIX. Эта система является иерархической совокупностью
	  каталогов и других файлов, имеющей древоподобную структуру. Структура имеет единственную корневую директорию (root) - /.
	   Для навигации по файловой системе можно использовать команду cd с аргументом пути, куда нужно переместиться.
	    Можно использовать два вида путей: относительный, абсолютный. Первый указывает положение относительно текущей директории, второй относительно корневой директории.
	     Для вывода абсолютного пути до текущей директории можно использовать команду pwd.
	      Для вывода файлов в директории можно использовать команду ls.
